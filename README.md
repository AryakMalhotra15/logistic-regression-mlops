# Logistic Regression MLOps Project

## Overview
This project demonstrates a simple Logistic Regression model using Scikit-learn.

## Dataset
Breast Cancer dataset from sklearn.

## Steps
- Data loading
- Train-test split
- Model training
- Evaluation
- Model saving using joblib

## Accuracy
~95-98% (depending on split)

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run notebook
